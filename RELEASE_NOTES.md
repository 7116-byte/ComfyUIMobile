# v0.1.0

首个公开版本。

- 原生 Android 简体中文界面，采用“工作流 → 参数 → 结果 → 任务”结构，不在主界面展示节点画布。
- 支持可信局域网连接与扫描、WebSocket 进度、队列和历史监控。
- 读取服务器真实工作流并生成纵向参数表单，常用参数显示中文名称。
- 支持工作流导入、导出、复制、改名、移动、删除、冲突保护和参数布局保存。
- 支持提示词历史、图片/视频上传、生成、图片查看、视频流式播放、保存和分享。
- 支持 GitHub Release 更新检查，并校验 SHA-256、包名、版本号和签名证书。

安全提醒：本应用只面向可信局域网中的无认证 ComfyUI，不要把 8188 端口暴露到公网。
