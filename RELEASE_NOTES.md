# v0.1.4

- 修复从图片导入工作流失败：PNG 工作流元数据改由 App 直接读取，不再依赖隐藏网页跨域取文件；导入重名时自动创建副本。
- 点击工作流后自动进入参数页。
- 参数页只读取连接到有效输出的实际执行链，未连线、已绕过和整条停用的分支不再显示，也不会提交到服务器。
- 部件按真实依赖顺序排列；加载器、空 Latent、采样器、解码和保存节点不再按画布内部编号乱序。
- 服务器校验失败时定位、标红并自动展开具体部件，不再只显示一条无法判断来源的错误。
- 结果页新增“本地/云端”和“全部平铺/任务相册”；每次生成任务为一个相册，最新任务和媒体排在最前。
- 参数页可长按输出部件加入本地缓存白名单，设置页可停用或删除规则、清空缓存。
- 本地只缓存本 App 提交且命中白名单的输出节点；电脑浏览器提交的任务不会进入本地缓存。

安全提醒：本应用只面向可信局域网中的无认证 ComfyUI，不要把 8188 端口暴露到公网。
