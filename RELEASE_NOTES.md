# v0.1.6

- 修复任务完成后停在“部件 null”、结果页和本地缓存不刷新的问题。
- 连接编号改为每种连接颜色独立计数；同一输出分叉仍使用 `a/b/c` 后缀。
- 部件展开和收起改用常见的折叠图标。
- 兼容 ComfyUI 先发送成功事件、后写入历史的时序，批量输出会在历史落盘后完整同步。

# v0.1.5

- 增加 ComfyUI 0.28 的 `progress_state` 实时进度支持，顶部和参数页显示当前部件及百分比。
- 正在执行的参数部件显示绿色边框并自动滚动到屏幕内，执行过程会像网页端一样逐个部件移动。
- 生成完成、失败、中断和取消均有明确状态，并及时刷新结果和停止任务通知。
- 工作流列表改为：单击选择、双击进入参数、长按保存；导入后不再因单击自动跳页。
- 参数部件左右两侧新增彩色输入/输出连接点与数字标记；分叉连接以下游 `4a/4b/4c` 标记表示同属上游输出 `4`。
- 修复批次结果并发写入时本地相册可能只保留最后一张的问题；升级后刷新结果会补齐此前缺少的同批图片。
- 图片改用全屏图库查看，支持双指缩放、放大后拖动和左右滑动切换图片。

安全提醒：本应用只面向可信局域网中的无认证 ComfyUI，不要把 8188 端口暴露到公网。
