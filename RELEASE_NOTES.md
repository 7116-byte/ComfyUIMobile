# v0.1.3

- “导入”现在可选择包含 ComfyUI 工作流信息的 PNG、WebP、AVIF 图片，也继续支持 JSON。
- 参数页改为按真实节点分组：UNet 加载器、K 采样器等各自是一张可展开的设置卡片。
- 自动排除被绕过（mode 4）和永不执行（mode 2）的节点，整条停用流程不再占用参数页。
- 支持 rgthree 的 Fast Groups Bypasser 自定义开关；切换分组后重新读取节点状态并刷新参数列表。
- 修复同一节点内重名自定义控件可能改错目标的问题。
- 保留未启用分支已经保存的手机布局配置，来回切换流程不会丢失参数名称和分组。

安全提醒：本应用只面向可信局域网中的无认证 ComfyUI，不要把 8188 端口暴露到公网。
